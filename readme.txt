# The TestSuite is Written On "Windows Machine" File Path Structure is Of windows
Written With Python And Robot Framework
 
Steps to Run The test Suite
1) There are 3 Files "config.py","api_calls.py","msg_api.robot" 
   All The 3 files Should be in same Directory
2) Cmd To Run The Script 
   robot msg_api.robot

Note: If robot cmd is not found, Please install robotframework (pip install robotframework)
	or else check robot path is set in environment variable
